# Portfolio
Personal portfolio website developed as a long-term project, combining web development with cybersecurity-focused design, testing, and documentation.

This repository contains the source code for my personal portfolio website.

The portfolio is developed as a long-term cyber life project where I apply secure development practices while also focusing on clean, usable, and professional UI/UX design. It represents how I approach building public-facing systems with both security and user experience in mind.

## Objectives
- Build a professional portfolio website with a strong security-first foundation
- Apply security-by-design and basic threat modeling principles
- Design clear, accessible, and user-focused interfaces
- Treat the portfolio as a real-world public-facing application
- Continuously test, improve, and document both security and usability aspects

## Scope
- Frontend: HTML, CSS, JavaScript (UI/UX focused layout and interactions)
- Backend (optional): Node.js and Express

In progress
